# Mini Budget Tracker

A simple web-based budget tracker built with HTML5, CSS3, and JavaScript. Track your income and expenses, categorize transactions,
and see a running balance. Designed with modular JavaScript and clean code principles.

# Features
- Add income and expenses with descriptions and categories
- Calculate current balance
- View a summary of total income, total expenses, and transaction count
- Transaction list income and expenses


## Technologies
- HTML5
- CSS3
- JavaScript (ES6 module pattern)

## How to run / test
1. Open `index.html` in a modern web browser.
2. Enter the **amount**, **description**, and **category**.
3. Click **Add Income** or **Add Expense**.
4. The balance, summary, and transaction list update automatically.
5. Open the browser console (F12) to see logged results.
